# 🍫 Chocolate Sales Analysis – Dashboard & Data Exploration (R)

Ce projet analyse les ventes d’une entreprise fictive de chocolat à partir d’un dataset Kaggle.  
Il inclut le nettoyage des données, l’analyse exploratoire, la création de KPIs et un dashboard interactif développé avec **Shiny**.

---

## 📊 Objectifs du projet

- Explorer les ventes, profits et marges
- Identifier les produits les plus performants
- Analyser le comportement des clients
- Visualiser les ventes par localisation
- Construire un dashboard interactif professionnel

---

## 🗂️ Dataset

Le dataset utilisé provient de **Kaggle**.  
Il n’est pas inclus dans ce repository pour des raisons de licence.
